# A/B Test Analysis: Subscription Discount Offer

This project analyzes the results of an A/B test conducted for a mobile app subscription offer.

The goal of the test was to compare two subscription pricing variants and determine whether the discounted offer produced a statistically significant improvement in conversion rate.

## Test Context

Two groups of users were included in the experiment:

- **Group A:** Users were offered access to premium features for **$4.99**.
- **Group B:** Users were offered the same subscription with a **50% discount**.

The main metric analyzed in this project is the conversion rate from installation/paywall exposure to payment.
